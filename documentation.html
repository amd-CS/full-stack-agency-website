Documentation

Instructions

There are four pages on the website: Home, Tourist Attractions, Send Comment and Documentation.

Each page's name is pretty self-explanatory. On the "Send Comment" page, users can send comments
which are added to and stored in the database.

Features

- comments are stored in the database
- four pages
- form with client-side and server-side validation
- users can add rows records to the database by sending comments
- several uses of CSS transitions (hover effect over the navigation menu items, etc.)
- two @keyframes animations: background image animation, animation when hovering over
images on the "Tourist Attractions" page which hides the image and reveals the name of the place
- several uses of CSS Grid and FlexBox (items on the home page, "Tourist Attractions" page, etc.)
- JavaScript functionality: keyboard accessible menu (pressing the first letter of each menu items
makes the corresponding menu item focused)
- responsive, mobile-first design
- accessibility features: alt text for images, etc.
- HTML and CSS validation
